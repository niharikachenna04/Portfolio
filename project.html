AI IN AEC INDUSTRY

SECTION 04

PROJECTS & USE CASES

This section highlights the main projects I completed during the course and explains how AI tools can support construction workflows, documentation, estimating, communication, and decision-making in the AEC industry.

MINI AI AGENT

Mini AI Agent — Description and Tools Used

This project focused on creating a custom mini-agent that helps AEC professionals prepare clear and professional project documentation.

It supports tasks such as RFIs, meeting minutes, site visit reports, follow-up emails, and organized construction communication.

NEW ADDITION

Reviewing of Mini Agents

This project focused on reviewing and evaluating mini AI agents based on their usefulness, clarity, accuracy, and practical value for construction-related tasks.

Through this review, I learned how to compare AI outputs, identify limitations, and understand where human judgment is still important.

BOOK REVIEW GPT

Book Review GPT — Outcome and Relevance

This project used an AI-powered book summarizer to generate a clear and structured overview of The Lean Startup.

It showed how AI can quickly extract key ideas, summarize content, and improve learning efficiency.

ESTIMATING AI

Estimating with Togal AI

This project explored how AI can support quantity takeoffs from construction drawings.

Using Togal AI, I reviewed drawing types such as floor plans, reflected ceiling plans, and foundation plans to understand how automated takeoffs improve estimating workflows.

FINAL PROJECT

Final Project — Its AEC Significance

The final project connected AI learning with practical AEC applications by focusing on how intelligent tools can support project teams, reduce repetitive work, and improve communication.

It helped me understand AI as a professional support tool rather than just a technology trend.

PROJECT TAKEAWAY

AI becomes more valuable when it is connected to real project needs.

These projects helped me understand that AI can support AEC professionals when it is used with clear purpose, human review, and construction-specific knowledge.
